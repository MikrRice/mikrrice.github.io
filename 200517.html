Includes Initial setup, Component upgrade suggestions, General tips

First Steps

Right off the bat you're going to want to use the PC you have, with everything you have in it already.

As you have all the hardware needed to boot a PC, you just need an operating system to be able to use it with modern programs. Windows 10 is really the only viable option in my opinion. If you're feeling adventurous you can try a Linux distro instead, though Windows is supported by many more programs.

  • Option 1: Buy Windows 10 Home 64-bit
  • Option 2: Use Windows 10 LTSC 1809 64-bit
  • Option 3: Use a Linux distro

    • Choosing Windows

    The easiest option for Windows 10 is to buy it online or in the tech section of a local store (Best Buy, Walmart, Fred Meyer, etc.) If you buy it online you'll have options for a download with a product key, a mailed DVD, or a mailed USB Stick. The DVD version is available on Newegg for $109.99.

    Alternatively you can use Windows 10 LTSC 1809 64-bit. This is a version of Windows intended for use in long-term, enterprise-type depolyments, like store kiosks and the like. This is a little tricky to get up and running, but doesn't come with a lot of bloatware that Windows 10 Home does. It's also not officially licensed, so you have to run a hack tool once every 6 months to "re-license" the software. This option has some benefits, including being free. If you choose to go this route I can walk you through how to get a legit version of this online.

    Choosing Linux

    If you choose to go the adventurous route and use Linux you'll need to research which distro you'd like to use. I won't be much help in supporting this, but there are quite a few online communities that can help out. I've used Ubuntu LTS in the past, and Canonical has release a new stable version last month that will be supported until April 2030, so that'd be a good place to start.
